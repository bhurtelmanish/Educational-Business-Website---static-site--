Diploma in Computer Application

The Diploma in Computer Application course focuses on equipping participants with practical skills in utilizing various software applications. It emphasizes Adobe software proficiency, Microsoft Access database management, and the process of CD burning. Furthermore, participants will gain insights into installation processes, virus scanning techniques, and effective utilization of email and internet resources.

The curriculum covers an array of topics essential for practical computer application skills:

  • - Adobe PageMaker 7.0
  • - Adobe Photoshop CS3
  • - CorelDraw X4
  • - Microsoft Access 2007
  • - Install/ Remove Program
  • - Windows Moviemaker
  • - CD Burning
  • - Virus Scanning
  • - E-Mail & Internet
